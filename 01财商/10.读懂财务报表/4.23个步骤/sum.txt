1.净利润 = （期末盈余公积 + 期末未分配利润）（资产负债表） + 本期已实施分红 （现金流量表）- （期初盈余公积 + 期初为分配利润）（资产负债表）

2.资产负债表中的现金就是从合并现金流量表中的期初现金及现金等价物余额 + 现金及现金等价物净增加额

3.查看本期分红多少钱就去合并现金流量表的股息分红

第一步：准备好目标公司的年报
1）下载公司5- 10年的年报 + 招股说明书
2）下载同行业前三名公司5-10年的年报

第二步：先看目标公司近期的年报
1）先从合并资产负债表看起

第三步：把合并资产负债表各个科目看一遍，标记异常科目
1）同比增长或下降30%的科目标记一下，回头重点看，并查明原因
2）像这种占总资产小于3%的科目，出现较大的异常变动也是可以忽略不计的，影响有限

第四步：搜索异常科目，查明原因
1）比如其他流动资产出现超过30%的变动，搜索“其他流动资产”可以看到海天味业资金购买了银行理财，进行货币资金管理也是可以接受的
2）比如搜索“预收款项”出现超过30%的变动，搜索“预收款项”可以看到是因为年末经销商提前打款造成的，竞争力强的表现
3）资产项目造假高发科目：
应收票据
应收账款
其他应收款
预付款项
存货
在建工程
长期待摊费
4）负债项目造假高发科目：
短期借款
应付票据
应付账款
其他应付款

第五步 看总资产，判断公司实力及扩展能力
总资产金额大公司实力强，同比增长快扩张能力强
 
第六步：看资产负债率，判断公司的债务风险
>60%的公司 淘汰

第七步：看有息负债和货币资金，排除债务风险
对于资产负债率大于40%的公司，要看它的货币资金是否大于有息负债，小于有息负债则淘汰
有息负债总额 = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券 + 长期应付款 + 应付利息

第八步：看“应收预付”和“应付预收” 判断公司的行业地位 看连续五年
行业地位强的留下，行业地位弱的淘汰
1）这个至少要看五年的数据
2）对比行业的前三位老大五年的数据，行业地位弱的淘汰

第九步：看固定资产，判断公司的轻重
固定资产 + 在建工程 + 工程物资 ／ 总资产 > 40% 淘汰，风险高，保持竞争力的成本大

第十步：看投资类资产，判断公司专注程度
包括两种：
与主业无关的投资类资产
1）以公允价值计量且变动计入当期损益的金融资产
2）可供出售金融资产
3）持有至到期投资
4）投资性房地产
5）与主业无关的长期股权投资
与主业有关的投资类资产
1）长期股权投资
注意：
1）优秀的公司一定是专注于主业的公司，与主业无关的投资类资产／总资产越低越好，最好是0
2）将与主业无关的投资类资产／总资产 > 10% 说明该公司在自己行业找不到更赚钱的标的了，淘汰

第十一步：把利润表盒现金流量表各个科目看一遍，标记异常科目
同比增长或下降30%的科目标记一下，回头重点看，并查明原因

第十二步：搜索异常科目，查明原因
直接搜索科目名称即可

第十三步：看营业收入，判断公司行业地位及成长能力，看连续五年
1）营业收入金额较大，且“销售商品，提供劳务收到的现金”/营业收入 > 1.1,行业地位高，产品竞争力强
2）营业收入增长率 > 10% 成长率较好
所以营业收入金额较大，且“销售商品，提供劳务收到的现金”/营业收入 < 1, 营业收入增长率小于10%的公司淘汰
3）看连续五年

第十四步：看毛利率，判断产品竞争力 看连续五年
<40% ，淘汰
1）如果一家公司比行业老大的毛利率高，很可能在造假
2）40%左右且逐步增强也可以哦

第十五步：看费用率，判断公司成本管控能力
1)优秀的公司 费用率／毛利率 < 40%
2)费用率／毛利率 > 60% 淘汰

第十六步：看主营利润，判断公司的盈利能力及利润质量 看连续五年
1）主营利润 < 0 淘汰
2）主营利润率<15% 淘汰
3）主营利润／利润总额 < 80% 淘汰
注：
主营利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用（为正数时）
主营利润率 = 主营利润 ／ 营业收入

第十七步：看净利润，判断公司经营成果及含金量
净利润越大越好
1）净利润 < 0 淘汰
2）优秀的公司“净利润含金比例”会持续大于100%，净利润现金含量 = 经营活动产生的现金流量净额总额／净利润
3）过去五年的净利润现金含量比率 = 过去五年的“经营活动产生的现金流量净额总额“／过去五年的“净利润”总和 < 100% 淘汰
 
第十八步：看归母净利润，判断公司自有资本的获利能力
1)优秀公司ROE持续大于15% ROE = 净利润／所有者权益
2）ROE < 15% 淘汰 （直接搜索“净资产收益率”）

第十九步：看经营活动产生的现金流量净额，判断公司造血能力
越大越好
1）公司经营活动产生的现金流量净额 > 固定资产折旧 + 无形资产摊销+借款利息+现金股利 可以
2）经营活动产生的现金流量净额 持续小于 固定资产折旧 + 无形资产摊销+借款利息+现金股利 淘汰

第二十步：看“固定资产，无形资产和其他长期资产支付的现金”，判断公司未来的成长能力
1）固定资产，无形资产和其他长期资产支付的现金／经营活动现金流量净额 10% - 60%，连续2年高于100%或低于10%的公司淘汰

第二十一步：看“分配给普通股股东及限制性股票持有者股利支付的现金”，判断公司品质
分红金额就去看合并现金流量表的筹资活动产生的现金流量里面
1）分红率 < 30% 淘汰（直接搜索“现金分红的数额”）

第二十二步：看三大活动现金流量净额的组合类型，选择最佳的公司
1）优秀的公司 正负负 ／ 正正负
2）连续2年为其他类型的公司淘汰

第二十三步：看“现金及现金等价物的净增加额”，判断公司的稳定性
1）现金及现金等价物的净增加额 < 0 淘汰
